# OpenKB Skill

Optional Hermes skill for using OpenKB as a compiled markdown knowledge base.

This skill works alongside the `openviking` memory provider:

- OpenViking remains the only external memory provider
- OpenKB handles explicit KB workflows
- the OpenKB bridge command is shared with the OpenViking provider for
  optional recall/writeback integration

See:

- `optional-skills/research/openkb/SKILL.md`
- `optional-skills/research/openkb/scripts/openkb_bridge.py`
- `plugins/memory/openviking/README.md`
